# Geheimzahl erraten (C#)

Dieses Konsolenprogramm generiert eine geheime Zahl zwischen 1 und 100 und lässt den Nutzer versuchen, diese zu erraten.  
Nach jedem Versuch gibt das Programm Hinweise („zu hoch“ oder „zu niedrig“) und zählt, wie viele Versuche benötigt wurden.

---

## Funktionen

- Zufällige Zahl zwischen 1 und 100 wird erzeugt  
- Eingabe über die Konsole  
- Hinweise:  
  - „Zu hoch!“  
  - „Zu niedrig!“  
- Zählt automatisch die Anzahl der Versuche  
- Erfolgsmeldung mit Auswertung  

---

## Beispielablauf

```
Erraten Sie die geheime Zahl!

50
Zu niedrig!
Versuchen Sie es erneut!

75
Zu hoch!
Versuchen Sie es erneut!

68
Geschafft!, Geheimzahl erraten.
Sie haben 3 Versuche gebraucht.
```
